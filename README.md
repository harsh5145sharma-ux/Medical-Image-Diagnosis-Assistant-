# 🏥 Medical Image Diagnosis Assistant

An AI-powered web application that analyzes medical images and predicts potential diseases using Deep Learning and Computer Vision techniques.

---

## 🚀 Project Overview

Medical Image Diagnosis Assistant helps users upload a medical image and receive an AI-based diagnosis prediction along with a confidence score.

The system automates the initial image analysis process and demonstrates how Artificial Intelligence can support healthcare professionals in medical image interpretation.

---

## ✨ Key Features

✔ Upload medical images

✔ Automatic image preprocessing

✔ Disease prediction using Deep Learning

✔ Confidence score generation

✔ User-friendly web interface

✔ Real-time prediction results
